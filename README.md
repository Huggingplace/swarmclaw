# SwarmClaw 🦀🤖

Welcome to the **SwarmClaw** core repository.

This repository contains the high-performance, Rust-native AI agent runtime designed for massive scale and Zero-Copy WASM execution.

## Repository Structure
- **`swarmclaw/`**: The core orchestrator and runtime binary. (See [swarmclaw/README.md](swarmclaw/README.md) for full details).
- **`swarmclaw-sdk/`**: The Rust SDK for building Zero-Copy FlatBuffers WASM skills.
- **`skills-library/`**: The standard library of securely sandboxed WebAssembly tools (Filesystem, Shell, etc.).
- **`docs/`**: Core protocol and architecture documentation (including the WebRTC Minion Protocol).
